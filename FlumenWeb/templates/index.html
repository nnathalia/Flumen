Flúmen

Projeto

O objetivo do projeto é desenvolver uma estação meteorológica inteligente com Raspberry Pi para coletar e analisar dados climáticos. Ela fornecerá informações precisas sobre variáveis climáticas para monitoramento, previsão e alerta de fenômenos, além de apoiar o planejamento estratégico em várias áreas.

Estação

Estações meteorológicas coletam dados climáticos diários. Existem versões convencionais, que requerem coleta manual, e automáticas, que usam sensores para coleta contínua e sem intervenção. Estações automáticas são mais eficientes e economizam tempo. Com a tecnologia e IoT, as estações inteligentes permitem acesso remoto e em tempo real aos dados.

Desenvolvimento

O projeto desenvolverá uma estação meteorológica inteligente com Raspberry Pi, equipada com sensores para medir umidade, temperatura, vento e pressão. Os dados serão processados e acessíveis remotamente em tempo real via internet. Além disso, foi desenvolvido o sistema web chamado Flúmem para acompanhar e visualizar os dados fornecidos pela estação. O objetivo é oferecer uma solução precisa e prática para o monitoramento e planejamento climático.
